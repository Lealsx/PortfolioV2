# Portfolio V2

Meu portfólio pessoal — versão 2.0. Desenvolvido do zero com foco em design clean mantendo a estética hacker.

## Sobre

Redesign completo do portfólio anterior com foco em legibilidade, espaçamento e experiência do usuário. O tema hacker foi mantido de forma mais sutil — tipografia monospace, fundo escuro, acento verde — sem os excessos de animação da versão anterior.

## Seções

- **Hero** — apresentação com typing animation no cargo
- **Sobre** — terminal estilizado com stack e informações pessoais
- **Projetos** — cards com tecnologias e links para GitHub e demo
- **Contato** — links para GitHub, LinkedIn e e-mail

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (Intersection Observer, typing animation)
- Google Fonts — Fira Code + Inter

## Funcionalidades

- Navbar fixa com active link automático via scroll
- Typing animation no cargo (loop entre funções)
- Scroll reveal nas seções
- Menu mobile responsivo
- Grid de fundo sutil com radial gradient

## Como visualizar

Abra o `index.html` diretamente no browser ou use o Live Server do VS Code.
